# roo_display API docs

This folder is used for generated API documentation.

It may also contain source markdown for local design notes related to
`roo_display` internals.

- Doxygen output is configured to go to docs/doxygen/html.
- GitHub Pages should publish docs/doxygen/html.

To build locally:
1) Install Doxygen.
2) Run `doxygen` from the repository root.
